# FCSC 2019 Petit Poucet

Traquez le petit poucet !



Auteur : [Cryptanalyse](https://twitter.com/Cryptanalyse)

Origine : [Petit Poucet](https://hackropole.fr/fr/challenges/misc/fcsc2019-misc-petit-poucet/)

> http://localhost

#### Tentez
